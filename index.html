Blok 1 2019

Ik heb voor het eerste blok van de 5e klas mijn eigen project bedacht met Femke Wenneker: Gamedevelopment in Python.
Waarom heb ik dit eigen project bedacht? Mij leken de aangeboden projecten niet zo leuk. Ik heb zelf nog nooit met python gewerkt, dus het leek mij leuk om daarme door middel van een project bekender te raken.
Ook vind ik het leuk om games te bouwen, dus leek mij het een goed idee om python en gamebuilding te combineren.
Ik ga dus tijdens dit project kennis maken met python, en het proberen goed onder de knie te krijgen door een game te bouwen.
Wegens mijn gebrek aan kennis over python, heb ik met Femke Wenneker overlegd, die een soortgelijk project gaat doen.

Het concept

Het idee is om een text-based roleplaying game te maken waarbij de speler steeds een keuze krijgt tussen een aantal acties, en zo invloed heeft op de verhaallijn van het spel.
Voor de module ga ik een stuk code schrijven die de speler de mogelijkheid geeft zich tussen de locaties in het spel te bewegen, wat qua beschrijvingen van de locaties ook uitgewerkt gaat worden.
Verder heb ik het idee van een aantal verhalen die ik wil uitwerken en moet de speler natuurlijk verschillende acties kunnen ondernemen.
Wat ik denk dat daarna ook nog mogelijk moet zijn, is het aantal locaties en de verhaallijnen die daarbij horen uitbreiden en dingen toevoegen als wapens en vijanden om tegen te vechten
Ik vind het moeilijk om in te schatten hoeveel tijd het bovenstaande kost, maar ik heb anders nog meer ideeen die ik zou kunnen uitwerken


Ik zal de code hiervan schrijven in Python. Hiervoor heb ik Python gedownload op mijn laptop

Begeleiding


Bij dit project ga ik geen of nauwelijks begeleiding nodig hebben, want het meeste kan ik opzoeken en als ik ergens niet uitkom, kan ik met Femke Wenneker overleggen, want zij gaat dus ook dit project doen, maar we maken elk onze eigen game.
